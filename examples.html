Netmatters Reflection:
Mouse-over popup

Sassy Cascading Style Sheets

$arrow-width: 100px;
$box-width: 240px;
.arrow {
    position: absolute;
    color: $color-tuna;
    height: 0;
    width: $arrow-width;
    bottom: calc(100% + 15px);
    left: calc(50% - (#{$arrow-width} / 2));
    border-left: ($arrow-width / 2) solid transparent;
    border-right: ($arrow-width / 2) solid transparent;
    border-top: ($arrow-width / 2) solid;
}
.box {
    position: absolute;
    background-color: $color-tuna;
    width: $box-width;
    bottom: calc(100% + (#{$arrow-width} / 2) + 15px);
    left: calc(50% - (#{$box-width} / 2));
}

Style and position a pair of objects for showing content in.

Manipulate the way that borders intersect to create a triangle by turning the other sides transparent.

Uses the calc() function to set the position of the triangle above and centered over its relative object, and the content box adjacent directly above it.

Netmatters Reflection:
Assigning styles to each icon

Sassy Cascading Style Sheets

@mixin icon($service, $font-size) {
  &::before {
      font-size: $font-size;
      content: map-get($service-icons, $service);
  }
}                

@each $service, $color in $service-colors {
    &.#{$service} {
        i {
            @include icon($service, 2em);
        }
        > i::before {
            border-radius: 50%;
            width: 65px;
            height: 65px;
            background: $color;
            color: $color-white;
        }
        &:hover {
            top: -10px;
            background-color: $color;
            color: $color-white;
            i::before {
                color: $color;
                background-color: $color-white;
            }
        }
    }
}

The icon @mixin takes a service name as the key to map each icon.

An each loop then iterates through all of the different services setting their styles, each with their own color and icon

This styles 7 different icons with the same section of code rather than having to repeat it.
